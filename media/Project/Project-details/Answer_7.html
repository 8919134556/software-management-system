Show transcribed image text A 15-element uniform linear scanning array has N4 spacing between elements. What is the simplified expression for the Array Factor for this array (5 pts) and what is ? 스 a)

ANSWER

Given N = 15 Array factor A F(theta, Phi) = sin(N Psi/2)/N sin (Psi/2) AF (theta, Psi) = sin (15 Psi/2)/15 sin (Psi/2) where Psi = beta d cos theta + Phi d = lambda/4, beta = 2 pi/lambda Phi = 0 in uniform linear scanning Psi = 2 pi/lambda lambda/4 cos theta + 0 Therefore Psi = pi/2 cos theta so AF (theta, Psi) = sin (15 pi/4 cos theta)/15 sin (pi/4 cos theta)
